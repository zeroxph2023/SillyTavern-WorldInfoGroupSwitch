# WorldInfo Group Switch

SillyTavern extension for managing world info groups with batch toggle.

Forked from LenAnderson's WorldInfoSwitch, completely rewritten by zeroxph2023.

## Features

- Right-click WI drawer icon for mode selection
- Original mode: WorldInfoSwitch-style interaction
- Group mode: group list + expandable book list
- Group CRUD (add/rename/delete)
- Batch toggle per group
- Per-book group assignment dropdown

## Version History

- v3.0.0: Complete rewrite as GroupSwitch (replaces old Presets fork)
- v2.x: Abandoned tree-panel approach (archived)
